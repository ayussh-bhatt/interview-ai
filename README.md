# AI Interview Prep Platform

Full-stack AI-powered mock interview platform.

## Structure
- `frontend/` – React + Tailwind UI
- `backend/` – Node.js + Express + Prisma + Gemini AI

## Features
- AI-generated interview questions
- Video & text interview responses
- Interview-level AI evaluation
- PDF-style interview reports
- Dashboard with past interviews

## Tech Stack
- React
- Node.js / Express
- Prisma + PostgreSQL
- Firebase Auth
- Google Gemini API
